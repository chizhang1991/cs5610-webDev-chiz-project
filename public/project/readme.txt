Project basic requirements
1. at least two user types
2. at least one third party API call
3. when submitting your project, please write a readme.txt file to explain how to use your app. a credential should be also included.
4. using bootstrap template
5. using node js and express js for API calls
6. using mongo db to store data
7. app should be accessible from your heroku URL
8. using angular js
9. using passport js to manage user login
10. login page, register page are required.
